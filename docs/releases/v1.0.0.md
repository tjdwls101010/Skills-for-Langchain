The first public release packages the evidence-built LangChain ecosystem skill as an installable Claude Code plugin and marketplace.

## Install

```bash
claude plugin marketplace add tjdwls101010/Skills-for-Langchain@v1.0.0
claude plugin install skills-for-langchain@skills-for-langchain
```

Restart Claude Code or run `/reload-plugins`. The explicit invocation is `/skills-for-langchain:langchain`.

## Highlights

- Python guidance for current LangChain 1.x, LangGraph 1.x, and the Deep Agents SDK.
- 6 measured LangChain deltas, 4 LangGraph deltas, and 16 Deep Agents content topics.
- Progressive disclosure through one short skill and two task-shaped references.
- Public baseline tasks, grader records, current-file hashes, and snapshot provenance.
- Standard open-source documentation, contribution, security, support, and issue workflows.

## Validation

- 38 baseline tasks: 27 included probe tasks and 11 measured-correct regression guards. C1 and C2 combine into one Deep Agents content topic, yielding 26 public content topics.
- Final recorded state: 27/27 included probe tasks correct and 11/11 exclusions preserved.
- Strict generated-file evidence was 35/38 before targeted repair and the independent current-file residual gate.
- Final residual gate: 6/6 independent grader verdicts `correct`, synthesis pass, no material residuals.
- Claude plugin and marketplace validation passed in strict mode.
- Harness validation passed with zero errors and zero warnings.

The evidence is layered; it is not represented as one identical-protocol 38/38 run. Headless E2E was skipped and is not claimed as release evidence.

## Compatibility boundary

- Python only.
- Verified against an official-documentation snapshot drafted around April 2026.
- Includes preview, private-preview, beta, and version-gated APIs.
- Review installed package signatures and current official docs for versions substantially newer than the snapshot.
